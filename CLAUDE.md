# FundFlow

初期投資額・月次積立・年利率・運用年数を入力するだけで、将来の資産推移をグラフと表で確認できる**静的 Web アプリ**。

> ⚠️ **アーキテクチャについて**
>
> 現在は **HTML / CSS / JavaScript（Vanilla）+ Chart.js** で構成されていますが、このテック スタックは**今後変わる可能性があります**。  
> フレームワーク化・ビルドツール導入・型安全性向上など、実装過程で大きな設計変更が起こる可能性があります。  
> タスク着手前に現在の構成を確認し、ドキュメントの更新が必要な場合は必ず CLAUDE.md を修正してください。

## 計算ロジック

### 複利（Compound Interest）

年初に1年分の積立額を加えてから年利を掛けます（年次複利）。

```
各年末資産 = (前年末資産 + 月次積立額 × 12) × (1 + 年利率)
```

### 単利（Simple Interest）

利息は常に元本にのみ発生し、利息が再投資されません。

```
利息 = 初期投資額 × 年利率 × 年数
     + 各年の積立額 × 年利率 × 残存年数（加重平均）
```

## 行動原則

- **既存コードを読まずに書かない** — 計算ロジックとビューの結合度を理解してから変更する
- **ブランチで作業する** — `main` への直接コミット禁止、ブランチ名: `feature/<作業内容>` または `claude/<作業内容>`
- **3ステップ以上のタスクは Plan モードで開始** — スコープ・影響範囲を明確にしてから実装
- **変更後は必ずブラウザで動作確認** — 複利計算・グラフ描画・UI が正常に機能しているか確認
- **実装完了後は CLAUDE.md を更新** — アーキテクチャ・ファイル構成に変更があれば記述
- **アーキテクチャ変更は事前相談** — フレームワーク導入・ビルドツール追加時は必ず相談してから進める

### よくあるバグ

| 症状 | 原因 | 対処法 |
|------|------|--------|
| グラフが表示されない | Chart.js CDN の遅延読み込み | DevTools > Network で CDN の応答確認 |
| 計算結果がおかしい | 月次利率の計算誤り | `monthlyRate = annualRate / 12 / 100` を確認 |
| スマホで崩れる | CSS メディアクエリの不足 | `max-width: 768px` のクラスを追加 |
