# FundFlow — かんたん投資シミュレーター

> 初期投資額・月次積立・年利率・運用年数を入力するだけで、将来の資産推移をグラフと表で確認できる静的Webアプリです。

**[→ シミュレーターを開く](https://yoyoyo-pg.github.io/fundflow/)**

---

## 機能

| 機能 | 説明 |
|------|------|
| 初期投資額 | 運用開始時の一括投資金額 |
| 毎月の積立額 | 毎月定額で追加投資する金額 |
| 年利率 | 期待する年間利回り（%） |
| 運用年数 | シミュレーション期間（1〜50年） |
| 複利 / 単利 切替 | 計算方式を選択可能 |
| 資産推移グラフ | 元本・最終資産を折れ線グラフで可視化 |
| 年別内訳テーブル | 各年の元本累計・運用益・資産合計を一覧表示 |

## 計算方式

### 複利（Compound Interest）

毎月、前月の資産に積立額を加えてから月次利率を掛けます。

```
月次利率 = 年利率 ÷ 12
各月末資産 = (前月末資産 + 月次積立額) × (1 + 月次利率)
```

### 単利（Simple Interest）

利息は常に元本にのみ発生し、利息が再投資されません。

```
利息 = 初期投資額 × 年利率 × 年数
     + 各年の積立額 × 年利率 × 残存年数（加重平均）
```

## 技術スタック

- **HTML / CSS / JavaScript**（バニラ、フレームワーク不使用）

- **GitHub Pages** — ホスティング

## ローカルで起動する

```bash
git clone https://github.com/yoyoyo-pg/fundflow.git
cd fundflow
# 任意の静的サーバーで開く（例）
npx serve .
# → http://localhost:3000
```

または `index.html` をブラウザで直接開くだけでも動作します。
